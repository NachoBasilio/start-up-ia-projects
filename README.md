# Start-up IA Projects

Guía base para arrancar documentación operativa de agentes de IA en cualquier proyecto.

## Qué trae este repo

- `start-up.md`: protocolo paso a paso para analizar un repo y crear `AGENTS.md` + skills sin humo.
- `Skills/skill-creator.md`: guía traducida para diseñar skills nuevas.
- `Skills/skill-sync.md`: guía traducida para mantener sincronizados metadatos y auto-invocación.

## Cómo usarlo

1. Copiá `start-up.md` al proyecto donde vas a trabajar.
2. Ejecutá la Fase 1 completa (análisis real del repo).
3. Diseñá `AGENTS.md` por capas (Fase 2).
4. Creá skills accionables (Fase 3) usando las referencias en `Skills/`.
5. Validá coherencia (Fase 4).
6. Corré validación técnica del stack activo (Fase 5).

## Regla clave

Para cualquier lenguaje o stack:

- siempre definir y ejecutar un linter o validación equivalente,
- correr validaciones antes de cerrar cambios y antes de commitear.

## Nota

Esta base es genérica a propósito. La IA debe ajustar comandos, paths y herramientas al contexto real del proyecto objetivo.
